# ArtemHP Theme
Technologies: css3, html5, javascript, grunts, sass/scss + compass, bower, node.js, haml

## Requirements

  * Ruby 2.0.0
  * SASS 3.3
  * HAML 4.0.5
  * node.js 1.3.14
  * Compass 0.12.3
  * Bower 1.3.0

## Quickstart

Install requirements soft described above.

Then navigate to the direcotry with project and just run the following commands:

```bash
npm install
bower install
grunt
```
